Utility Tool

My personal take on the popular Swiss Army Knife

Background & Build

I worked on this project to primarily familiarize myself with animation within SolidWorks. My utility tool design consists of 12 unique components that are listed below. The corkscrew design in particular was unique in the sense that it required the helix/spril sketch feature along with a sweep extrude tool to create. These are SolidWorks features that I got a change to work with for the first time with this design.

Frame and Dividers:
  • Front Cover
  • Back Cover
  • Pin
  • Separator 1, 2 & 3 (3 unique separator parts)
Tools:
  • Large Knife
  • Saw Knife
  • Ruler
  • File
  • Bottle Opener
  • Corkscrew

The premise of the design is simple. The pin components act as both the pivot locations of all tools as well as guides for the separators. The pins also get press fitted into the front and back cover allowing for a tool-free assembly. Along with designing each of these components, I also created part drawings for each individual component along with the assembly.
